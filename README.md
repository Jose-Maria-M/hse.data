INCIDENTES POR AREA Y SEVERIDAD

En este proyecto se realiza un analisis de datos en materia de seguridad, higiene y medioambiente utilizando python, para identificar patrones en incidentes laborales.

OBJETIVO: 
Identificar las areas con la mayor frecuencia de incidentes y evaluar la severidad de los eventos. 

Análisis realizado

El script realiza los siguientes pasos:

Carga de datos
Se importa un archivo Excel con información de incidentes.

Exploración inicial
Visualización de las primeras filas del dataset

Identificación de:

Áreas
Tipos de incidentes
Análisis de frecuencia
Conteo de incidentes por área
Identificación del área con más accidentes
Análisis de severidad
Clasificación de incidentes por nivel de severidad
